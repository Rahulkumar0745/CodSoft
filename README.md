# CodSoft

CodSoft is a frontend repo containing three polished web projects: a calculator, a landing page, and a portfolio website. Each project is built with HTML, CSS, and JavaScript, and includes a live demo link for quick preview.

## Projects

### Calculator
- Folder: `Calculator`
- Description: A responsive web calculator with basic arithmetic operations and a clean interface.
- Demo: [Calculator Demo](https://rahulkumar0745.github.io/CodSoft/Calculator/)

### Landing Page
- Folder: `LandingPage`
- Description: A modern, responsive landing page designed for showcasing products or services.
- Demo: [Landing Page Demo](https://rahulkumar0745.github.io/CodSoft/LandingPage/)

### Portfolio
- Folder: `Portfolio`
- Description: A personal portfolio website to display projects, skills, and contact information.
- Demo: [Portfolio Demo](https://rahulkumar0745.github.io/CodSoft/Portfolio/)

## How to Use

1. Open the desired project folder.
2. Launch the `index.html` file in your browser.
3. Customize text, styling, and content as needed.
