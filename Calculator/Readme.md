# Calculator Project

A simple web-based calculator built with HTML, CSS, and JavaScript.

## Demo

Live Demo: [Calculator Demo](https://rahulkumar0745.github.io/CodSoft/Calculator/)

## Description

This project demonstrates a functional calculator that supports basic arithmetic operations like addition, subtraction, multiplication, and division. It features a clean interface and responsive layout for desktop and mobile.

## Features

- Basic arithmetic operations: `+`, `-`, `*`, `/`
- Clear (`C`) and delete functionality
- Responsive design
- Keyboard input support (if implemented)

## Files

- `index.html` - Calculator user interface
- `style.css` - Styling for layout and buttons
- `script.js` - Calculation logic and event handling

## Usage

1. Open `index.html` in your browser.
2. Click buttons or use keyboard input to enter numbers and operators.
3. Press `=` to see the result.
4. Use `C` to clear the calculator.
